一个模仿 Windows 任务管理器的软件

关于

这是一个适用于 Windows 的任务管理器软件，模仿了 Windows 10 的任务管理器，在其基础上添加了一些附加的功能， 可以说是普通任务管理器的加强版（实际上是垃圾版），当然，还有很多功能没有完善。

本软件开源，遵守 MIT 协议，如果你对这个软件感兴趣，欢迎访问 github 下载软件源码研究（请原谅我的代码水平糟糕）。
